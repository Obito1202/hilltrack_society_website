# Hilltrack Society — your standalone website

Your standalone, GitHub-ready NGO website lives in the **`/hilltrack-site/`** folder.

👉 Open **[`hilltrack-site/README.md`](./hilltrack-site/README.md)** for the full guide.
👉 Open **[`hilltrack-site/SETUP.md`](./hilltrack-site/SETUP.md)** for a 5-step quick start.

Everything else in this repo (the `/backend/` and `/frontend/` folders) was the original Emergent prototype and is **not needed** for the standalone site. When you push to your own GitHub repo, you only need to push the `hilltrack-site/` folder.
